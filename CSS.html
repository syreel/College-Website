CSS

CSS Unit 20 P1 CSS is used in web pages in order to change the looks of the webpage. Html is used to create the layout of the webpage however it lacks the power to change the looks of the website in detail this is why we use CSS. CSS (Cascading Style Sheets) works by running over the html code for the website and applying styles to the elements in the webpage that are specified in the CSS file. You can use multiple CSS files on one webpage and these are overlaid on top of each other with the most recent one being higher priority and replacing styles layer down on past files. CSS can be used in multiple different methods such as in-line, in-line CSS is where you apply CSS directly to the element where it is referenced











Another method of CSS is Embedded this is used to tidy up the code a bit by moving all the CSS on a specific page into a style section this also helps make it easier to refer to multiple elements with a single CSS segment

The final method of using CSS on a webpage/website is external,external CSS is where all the CSS is stored in an external .css file then this file is referenced in one of your html files. This method is possibly the best method because it means that you can reference all the CSS in many webpages or html files. CSS Box Model Unit 20 P2 The CSS box model is based on the
tag it is used in order to structure websites in an organised way it is also used to make a responsive layout therefore when the screen size changes the layout of the website can be changed dynamically. The benefit of using divs over using a table is that tables cannot change as dynamically as divs can and also divs can be created and modified by CSS however tables are not as flexible. Margin – the margin is used to set a transparent gap around the div in order to distance it from nearby objects. The margin can be set separately and to different values on all four sides Border – the border is used to put a distinctive barrier between the div and other objects it creates a division in the website. This also has four sides that can be edited separately. Padding – the padding is used to create a gap between the content and the divs border this can make the text or other contents look more independent. This also has four sides that can be edited separately. Content – the content of a div can contain almost anything including nested divs therefore it can be used to create complex layouts very easily. Width – This is the width of the div it can be set as percentage, by pixel or by other measurements.
